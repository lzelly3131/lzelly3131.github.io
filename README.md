## Sobre mí

Desarrollador de software guatemalteco con más de 5 años de trayectoria construida desde el soporte técnico y el análisis de TI hasta el desarrollo web full stack. He mantenido y evolucionado sistemas ERP en producción, creado módulos e integraciones a la medida sin romper el estándar del producto, y apoyado migraciones de infraestructura para banca y empresas medianas en Guatemala. Combino una base sólida en PHP y JavaScript con experiencia práctica en frameworks modernos (React, Vue, Angular, Ionic), bases de datos relacionales y no relacionales, e infraestructura en la nube (AWS, Linux).

Soy autodidacta por naturaleza: priorizo soluciones permanentes sobre parches temporales y busco constantemente incorporar nuevas herramientas a mi flujo de trabajo — incluyendo asistentes de IA para desarrollo — para resolver problemas de raíz y automatizar procesos complejos con mayor eficiencia.
